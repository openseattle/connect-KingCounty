  2. LOOK

    3. This version of the design will use welcoming, warm colors. They will mainly be variants of brown. The reason for this is that brown is a color that conveys warmth, acceptance, and reliability. It inspires trust and it's the perfect combination for a site dedicated to helping people with some of the crises in life.

  3. WIRE FRAME

    Although there will be a number of pages, most will follow the list page style, where the page is mainly a list of icons to select. So, there is only one example of this style provided.

    Measurements are always important to get right. This site will use em as its base unit. Because...

    1. To make sure the page components resize correctly when the user zooms in or out. It is very frustrating to a user who can't see something, but can't use zoom to enlarge a page element.
    2. To make it easy to size things in relation to the text. Text is the core data piece of websites, so it makes sense to size thing according to that.

    The site is a little counter-intuitive in-that the navigation is on the bottom of the page, near the footer, and the main content is front-and-center. Because...

    1. On some low-tech devices, older browsers, etc, the navigation at top may obscure the content and bury it under hords of lists, sublists, and whatnot. If you've ever seen a site where the CSS didn't load, you know what I mean. If you haven't, open a pretty-looking website with navigation at-top, go into the code inspection, and delete the CSS links and embedded styles. So, if the content is first, this can be avoided.
    2. Having the content first also emphasizes it, so users get what they need as-fast-as-possible.
